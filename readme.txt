If you want to implement a "scroll to top" functionality on a webpage using this plugin, when clicked, will smoothly scroll the page to the top.

== Description ==

The Scroll-to-Top plugin is a lightweight and customizable solution for adding a "scroll to top" button to your website. This plugin enhances the user experience by providing a quick and smooth way for users to return to the top of the page.

=Usage:=
* The "scroll to top" button will automatically appear when the user scrolls down the page.
* Clicking the button will smoothly scroll the page to the top.

=Customization:=
* Unlimited  background colors.
* Unlimited  background hover colors.
* Position switcher (left or right).
* Position switcher (top or bottom).

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `back-to-top-wp` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings -> Back To Top.
